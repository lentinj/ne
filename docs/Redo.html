4.7.2 Redo ¶

Syntax: Redo [n]
Abbreviation: RE

redoes the last n actions undone by Undo (as long as you don’t take any actions that change the text between the Undo and Redo commands). If n is not specified, it is assumed to be one. You can only Redo actions that have been Undone. See Undo.
