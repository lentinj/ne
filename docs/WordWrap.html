4.8.7 WordWrap ¶

Syntax: WordWrap [0|1]
Abbreviation: WW

sets the word wrap flag. When this flag is true, ne will automatically break lines of text when you attempt to insert characters beyond the right margin. See RightMargin.

If you invoke WordWrap with no arguments, it will toggle the flag. If you specify 0 or 1, the flag will be set to false or true, respectively. A lower case ‘w’ will appear on the status bar if the flag is true.
