4.9.15 DelTabs ¶

Syntax: DelTabs [0|1]
Abbreviation: DT

sets the DelTabs flag. When this flag is set, a ‘d’ will appear on the status bar, and the BackSpace and Del keys will remove a tab’s worth of space characters if a TAB character could have occupied the same whitespace in the current line as the removed spaces. This is the deletion counterpart to the Tabs flag. See Tabs.
