4.12.13 NameConvert ¶

Syntax: NameConvert [0|1]
Abbreviation: NC

converts the current document’s name between relative and absolute path names. With no parameter it switches the current name from relative to absolute, or absolute to relative. With 1 it converts the relative path to absolute or leaves the absolute path unchanged. With 0 it converts the absolute path to relative or leaves the relative path unchanged.
