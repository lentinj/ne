4.3.5 CloseDoc ¶

Syntax: CloseDoc
Abbreviation: CD

closes the current document. If the document was modified since it was last saved, you have to confirm the action. If it is the only document, ne exits. Otherwise ne will select the most recently active next or previous document adjacent to the one you just closed as the new current document. See NextDoc, and PrevDoc.
