4.5.7 MatchBracket ¶

Syntax: MatchBracket
Abbreviation: MB

moves the cursor to the bracket associated with the bracket the cursor is on. If the cursor is not on a bracket, or there is no bracket associated with the current one, an error message is issued. Recognized brackets are ‘{}’, ‘()’, ‘[]’ ‘<>’, and ‘`'’. See AutoMatchBracket.
