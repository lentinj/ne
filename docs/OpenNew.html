4.2.2 OpenNew ¶

Syntax: OpenNew [filename]
Abbreviation: ON

creates a new document and loads into it the contents of the file specified by the optional filename string. This new document will inherit the macro, search, and replace strings from the current document.

If filename is unspecified, the file requester behaves the same as for the Open command; see Open.
