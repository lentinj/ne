5 Configuration ¶

In this chapter we shall see how the menus and the key bindings of ne can be completely configured. Note that menu and key configuration is parsed at startup time, and cannot be changed during the execution of the program. This is a chosen limitation.

We will also see how to override incorrect or missing file name extensions by comparing the contents of documents to patterns to determine virtual extensions.
