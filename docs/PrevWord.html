4.10.12 PrevWord ¶

Syntax: PrevWord [n][<|>]
Abbreviation: PW

moves the cursor to the left to the nth word beginning or ending. If the optional n argument is not specified, one is used. The optional ‘<’ or ‘>’ determines whether to move to the word beginning or ending, respectively. If unspecified, the left end (‘<’) is used.
