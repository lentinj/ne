4.11.5 DeletePrevWord ¶

Syntax: DeletePrevWord [n]
Abbreviation: DPW

deletes text from the current position through the first character of the n’th previous start-of-word. If the optional n argument is not specified, it is assumed to be one (in which case, if the cursor is in the middle of a word the effect is just to delete to the start of that word).
